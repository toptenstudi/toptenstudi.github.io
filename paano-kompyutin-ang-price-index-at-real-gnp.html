8. Taon 2006 2007 2008 2009 2010 Nominal GNP 10,500 11,208 12,223 13,505 14,622 Price Index Real GNP GROWTH RATE​


REAL GNP

Paano i compute ang Price index, una pumili ng base year. Sa problema ang ating gagamitin ay ang taong 2006 bilang base na taon. I divide ang nominal na GNP ng bawat taon sa nominal GNP ng base year 2006 at i multiply ng 100 upang makakuha ng price index sa bawat taon.

Paraan ng pag compute ng price index

Price index for 2006 = (P10,500 /P 10,500) x 100 = 100

Price index for 2007 = (P11,208 / P10,500) x 100 = 107.05

Price index for 2008 = (P12,223 / P10,500) x 100 = 116.41

Price index for 2009 = (P13,505 / P10,500) x 100 = 128.62

Price index for 2010 = (P14,622 / P10,500) x 100 = 139.26

Para naman I compute ang Real GNP narito ang FormulaReal GNP = (Nominal GNP / Price Index) x 100

Paraan ng Pagkuha ng real GNP

Real GNP for 2006 = (P10,500 / 100) x 100 = P10,500Real GNP for 2007 = (P11,208 / 107.05) x 100 = P10,480.83Real GNP for 2008 = (P12,223 / 116.41) x 100 = P10,494.26Real GNP for 2009 = (P13,505 / 128.62) x 100 = P10,505.92Real GNP for 2010 = (P14,622 / 139.26) x 100 = P10,506.56Paraan ng pag compute ng growth rateAng formula sa pag compute ng growth rate ay Growth Rate = ((Real GNP in Taon 2 - Real GNP in Taon 1) / Real GNP in Taon 1) x 100

Growth rate para sa 2007 = ((Real GNP in 2007 - Real GNP sa 2006) / Real GNP sa 2006) x 100

= ((P10,480.83 - P10,500) / P10,500) x 100= -0.18%

Growth rate para sa 2008 = ((Real GNP sa 2008 - Real GNP sa 2007) / Real GNP sa 2007) x 100

= ((P10,494.26 - P10,480.83) / P10,480.83) x 100= 0.13%

Growth rate para 2009 = ((Real GNP sa 2009 - Real GNP sa 2008) / Real GNP sa 2008) x 100

= ((P10,505.92 - P10,494.26) / P10,494.26) x 100

        = 0.11%

Growth rate para sa  2010 = ((Real GNP sa 2010 - Real GNP sa 2009) / Real GNP sa 2009) x 100

= ((P10,506.56 - P10,505.92) / P10,505.92) x 100= 0.01%

Price Index = (Nominal GNI / Real GNI) x 100
