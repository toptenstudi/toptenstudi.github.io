2. Four Arithmetic means between 8 and 23


Solve the above problems with arithmetic sequences.

Because what is being asked is four numbers between 8 and 23. So the arithmetic sequence has 6 sequences of numbers = 8,__, __, __, 23.

is known

The first term, or what is called a = 8, is

6th term, T6 = 23.

 

formula to find Tn

Tn = a + (n-1) d

With this formula, we enter the known T6 and a(first term) to find the difference or "d".

Tn = a+ (n-1) d

T6 = 8 + (6-1) d

23 = 8 + 5d.

5d = 23-8

5d = 15.

d = 3

After knowing the difference is 3, Then we can determine the 2,3,4, and 5th terms of the arithmetic sequence above.

Tn = Tn-1 + 3 is the formula.

T1 = 8

 

T2 = T1 + 3 = 8 + 3 = 11

T3 = T2 + 3 = 11 + 3 = 14

T4 = T3 + 3 = 14 + 3 = 17

T5 = T4 + 3 = 17 + 3 = 20.

 

So the arithmetic sequence from the 1st to the 6th term is 8,11,14,17,20,23,...

 

Arithmetic Sequence

An arithmetic sequence (Tn) is a sequence of numbers that has a fixed pattern based on addition and subtraction operations.

The arithmetic sequence consists of the first term (T1), the second term (T2), and so on up to n, or the nth term (Tn).

Each tribe has the same difference or difference. The difference in each term is called the difference, symbolized as "d". The first term, T1, is also symbolized as "a"

Number sequence: 0,5,10,15,20,25,...Tn

 

For example, the above is an arithmetic sequence that has the same difference, namely d = 5, and the first term is a = 0. The difference is obtained by subtracting each term. For example, the second term T2 minus the first term T1 gives b = T2 – T1 = 5 – 0 = 5. The value of d can also be obtained from the third term minus the second term and so on.
