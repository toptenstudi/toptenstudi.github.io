5. If RHOM is a rhombus, RH=6y+4, HO=5y+8, find the measurement of side OM​


Since RHOM is a rhombus, all four sides are equal in length. Let x be the length of side OM.

Then, by the properties of a rhombus:

RH = OM = 6y + 4 (opposite sides are equal)

HO = OM = 5y + 8 (opposite sides are equal)

Since RH = OM and HO = OM, we can equate these two expressions to get:

6y + 4 = 5y + 8

Simplifying this equation, we get:

y = 4

Now, we can substitute y = 4 into either RH = OM or HO = OM to find the length of side OM:

OM = RH = 6y + 4 = 6(4) + 4 = 28

Therefore, the length of side OM is 28.

14. - Given: m angle HER = 5y-26 m angle ROH = 6y - 48 Find angle ROH.​


ANGLE

Given:

m∠HER = 5y - 26

m∠ROH = 6y - 48

To find the value of ∠ROH, we need to have a specific value for y. If y is not given, we cannot find the exact value of the angle.

Assuming that we have a value for y, we can find the value of ∠ROH as follows:

m∠HER + m∠ROH = 180° [Angles on the same side of a transversal and inside the parallel lines add up to 180 degrees.]

Substituting the given values:

(5y - 26) + (6y - 48) = 180

11y - 74 = 180

11y = 254

y = 23

Substituting the value of y in the expression for ∠ROH:

m∠ROH = 6y - 48

m∠ROH = 6(23) - 48

m∠ROH = 138 - 48

m∠ROH = 90

Therefore, if y is equal to 23, then the value of ∠ROH is equal to 90 degrees.

Substituting the given measures, we get:

m∠H + m∠E + (5y - 26) = 180

Similarly, in triangle ROH, we have:

m∠R + m∠O + m∠H = 180

Substituting the given measures, we get:

(2y - 40) + m∠O + m∠H = 180

We are asked to find m∠ROH, which is the same as m∠O + m∠H. We can solve for this expression by combining the two equations above and isolating m∠O + m∠H:
