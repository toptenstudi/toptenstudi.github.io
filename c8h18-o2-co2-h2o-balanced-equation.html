7. 24-25. Balance the chemical equation below. Show evidences that the equation is balanced C2H6 + O2 + CO2 + H2O pa help​


Answer:

Balanced Equation: 2C₂H₆ + 7O₂ = 4CO₂ + 6H₂O.

Evidences:

number of C atoms left side = number of C atoms right side = 2number of H atoms left side = number of H atoms right side = 6number of O atoms left side = number of O atoms left side = 7

Explanation (how to balance the equation):

Count the number of atoms on both sides.

Left Side: C = 2, H = 6, O = 2Right Side: C = 1, H = 2, O = 2 + 1 = 3

Balance the atoms. To balance C, we need to multiply the number of C atoms on right side by 2. Similarly, multiply the number of H atoms on right side by 3. But, since we multiplied, we need to change the number of oxygen atoms on right side. So the oxygen atom on right side will be 2 × 2 + 3 × 1 = 7. Now, we need to balance the oxygen atoms.  By multiplying the number of O atoms on left side by 7/2, we've balanced the atoms. Our equation is now C₂H₆ + 7/2(O₂) = 2CO₂ + 3H₂O. Multiply both sides by 2, we yield our balanced equation: 2C₂H₆ + 7O₂ = 4CO₂ + 6H₂O.

17. Balance the following chemical equation, then answer the following question. C8H18(g)+O2(g)→CO2(g)+H2O(g) How many grams of oxygen are required to react with 16.0 grams of octane (C8H18) in the combustion of octane in gasoline?


Balancing the equation:

2C₆H₁₈ + 21O₂ -> 12CO₂ + 18H₂O

Note: Balance the Cs and Hs first, since they only appear at one substance in both sides.

Stoichiometry:

Take note of the molar masses of octane and oxygen gas.

C₆H₁₈ - 114.23 g/mol

O₂ - 32.00 g/mol

First, find the number of moles of octane in the given mass above.

16.0 g × [1 / 114.23 g/mol] = 0.140 mol C₆H₁₈

Now, use the mole ratio of octane to oxygen to obtain the number of moles of oxygen involved in the reaction.

N mol O₂ / 0.140 mol C₆H₁₈ = 21 mol O₂ / 2 mol C₆H₁₈

N / 0.140 = 21 / 2

N = 1.47 [mol O₂]

Finally, find the number of grams in 1.47 moles of oxygen gas.

1.47 mol × 32.00 g/mol = 47.04 g O₂

Further explanation  

The reaction equation is the chemical formula of reagents and product substances  

A reaction coefficient is a number in the chemical formula of a substance involved in the reaction equation. The reaction coefficient is useful for equalizing reagents and products.  

Equalization of chemical reactions can be done using variables. Steps in equalizing the reaction equation:  

1. gives a coefficient on substances involved in the equation of reaction such as a, b, or c, etc.  2. make an equation based on the similarity of the number of atoms where the number of atoms = coefficient × index between reactant and product  3. Select the coefficient of the substance with the most complex chemical formula equal to 1  

For gas combustion reaction which is a reaction of hydrocarbons with oxygen produces CO₂ and H₂O (water vapor). can use steps:  

Balancing C, H, and the last O atoms

Reaction :

C₃H₈ + O₂ ⇒ CO₂ + H₂O

1. Gives a coefficient

C₃H₈ + aO₂ ⇒ bCO₂ + cH₂O.

C ⇒ left = 3, right = b ⇒b = 3

H ⇒ left = 8, right = 2c ⇒2c=8 ⇒ c = 4

O ⇒ left = 2a, right = 2b+c ⇒ 2a=2b+c⇒2a=2.3+4⇒2a=10⇒a=5

The chemical equation becomes :

C₃H₈ + 5O₂ ⇒ 3CO₂ + 4H₂O

27. balance the following equations ? CH4 + O2 CO2 + H2O CaCl2 + AgNO3- Ca(NO3)2 + AgCl C2H6O + O2 CO2 + H2O


Further explanation  

Equalization of chemical reactions (balanced chemical equations) can be done using variables. Steps in equalizing the reaction equation:  

1. gives a coefficient on substances involved in the equation of reaction such as a, b, or c, etc.  2. make an equation based on the similarity of the number of atoms where the number of atoms = coefficient × index (subscript) between reactant and product  3. Select the coefficient of the substance with the most complex chemical formula equal to 1  

1. CH₄ + O₂ ⇒ CO₂ + H₂O

CH₄ + aO₂ -----> bCO₂ + cH₂O  

C: left = 1, right = b -> b = 1  

H: left = 4, right = 2c ---> 4 = 2c ---> c = 2  

O: left = 2a, right = 2b + c ---> 2a = 2b + c ---> 2a = 2.1 + 2 -> 2a = 4 -> a = 2  

The equation becomes:  

CH₄ + 2 O₂ -----> CO₂ + 2 H₂O  

2. CaCl₂ + AgNO₃ ⇒Ca(NO₃)₂ + AgCl

aCaCl₂ + bAgNO₃ ⇒Ca(NO₃)₂ + cAgCl

Ca : left = a , right = 1 ⇒ a=1

Cl : left = 2, right = c ⇒ c=2

Ag : left = b, right = c  ⇒ b=c = 2

NO₃ : left = b, right = 2 ⇒ b=2

The equation becomes:  

CaCl₂ + 2AgNO₃ ⇒Ca(NO₃)₂ + 2AgCl

3. C₂H₆O + O₂ ⇒ CO₂ + H₂O

C₂H₆O + aO₂ ⇒ bCO₂ + cH₂O

C : left=2, right =b ⇒ b=2

H : left =6, right = 2c ⇒ 2c=6 ⇒c=3

O : left =2a+1, right=2b+c ⇒2a+1=2b+c⇒2a+1=2.2+3⇒2a=6⇒a=3

The equation becomes:  

C₂H₆O + 3O₂ ⇒ 2CO₂ + 3H₂O
