15. How to find the value of k


potassium? Joke. There are many ways to find the value of K. Using the formulas:
y = kx (Direct Linear Variation)
y = [tex]k x^{2} [/tex] (Direct Square Variation)
y =[tex] \frac{k}{x} [/tex] (Inverse Variation)
y = kxz (Joint Variation)
y = [tex] \frac{kx}{z} [/tex] (Combined Variation)

For example in Direct Linear Variation;
a.) If y varies directly as x and y = 24 when x = 6, find the value of y when x = 15.

1st step:
(y = kx)
Since it is given in the first line that y = 24 and x = 6, substitute the y in the given formula to 24 and x to 6;
24 = k(6)

2nd step.
In order for us to find the value of k (or the variable K), we have to remove the number beside the given variable. How? Divide 6 to 6 and 24 to 6. It is important to keep in mind that if you divide the number beside a variable, you must also include the number to the other side.

it will look like this:

[tex] \frac{24}{6} [/tex] = [tex] \frac{k6}{6} [/tex]

3rd step.
divide it.

cancel out 6 and leave k (because 6/6 is = 1 and it is automatic that when there is a variable, the given number or whole number is 1.) Then divide 24 to 6. 24/6 = 4.

It'll look like this:

4 = K

Thus, the value of K is 4.
