2. Joy bought 5 pinipig Chunchesand I drumstick and Paid Php 97.Jeffrey paid 89 for 3 pinipig crunches and 2 drumsticks.How much did each pinipigand drumsticks cost?


Answer:

Each pinipig cost P15.00 and each drumstick cost P22.00.

Step-by-step explanation:

Let x be the cost of the pinipig and let y be the cost of the drumstick.

5x+1y=973x+2y=89

First, let us find the value of x. Eliminate y.

(5x + 1y = 97)2 = 10x + 2y = 194

3x + 2y =89 = 3x + 2y = 89

Subtract : 7x + 0 = 105

Solve : 7x/7 = 105/7

x = 15

Solve for y. Substitute x with 15.

5x+1y=97

5(15) + 1y= 97

75 + 1y = 97

1y/1 = 97- 75

y= 22

Kahulugan ng "Amoy Pinipig"

Ang katagang "amoy pinipig" ay isang halimbawa ng sawikain. Ang ibig sabihin nito ay mabango.

Ang "amoy pinipig" ay nangangahulugan ng mabango dahil ang pinipig ay kilala sa amoy na taglay nito. Ang pinipig ay ang mga tinustang hilaw na maligat na bigas na may mabango at nakakatakam na amoy.

Ang "amoy pinipig" ay kasalungat ng "amoy tsiko" na ang ibig sabihin naman ay mabaho.
