23. a50=21 + (50-1)7 find a50​


Answer and step-by-step explanation:

The given equation only needs to be simplified further.

Let's follow the PEMDAS rule which stands for Parentheses, Exponents, Multiplication and Division; Addition and Subtraction. Everything should be done from left to right.

a50 = 21 + (50-1)7 => subtract 1 from 50.

a50 = 21 + (49)7 => multiply 49 by 7.

a50 = 21 + 343 => add 21 and 343.

a50 = 364
