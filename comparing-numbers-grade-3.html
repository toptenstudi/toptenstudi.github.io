Ang Histogram is a graphical display of data using vertical bars of

different heights to represent the frequencies of the classes. It is

similar to bar graph but a histogram doesn't have any spaces

between and groups numbers into ranges.

It is used to summarize large data sets graphically and compare

measurements to specifications.

Histograms are a great way to show results of continuous data,

such as: height, weight, and other related topics. But if the given

data is in categories, we should use bar graph instead.

Note: Histogram should have the following parts:

a. Title

b. Vertical axis - this indicates the frequency occurred within an

interval.

c. Horizontal axis - this indicates the interval or scale of values

into which the measurement fit.

d. Bars - it has two important characteristics, the height and width.

The height represents the number of times the values within an

interval occurred. The width represents the length of the interval

covered by the bar, and it is the same for all bars.
