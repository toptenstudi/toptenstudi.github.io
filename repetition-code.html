7. A 5-digit pin code cannot begin with zero and the remaining digits have no restrictions. if repetition of digits is allowed, then what is the probability that the pin code begins with a 7 and ends with an 8?


Answer:   [tex]\frac{1}{90}[/tex]

Explanation:

Step 1: Determine the over-all number of possibilities.

Since a 5-digit PIN code cannot start with zero, therefore the first digit would be restricted by one less digit, and then the rest would be composed of any of the numerical digits from 0 to 9.

Let N be the number of possible PIN number outcomes. Therefore:

N = 9*10*10*10*10

N = 90000 PIN Numbers

Step 2: Determine the number of possible outcomes

To determine the number of possible outcomes, we must identify the constraints first. The constraint is that the pin code should begin in 7 and end at 8, therefore, we have to lock them in on the first and last position. So, the first and last slot should have no other possibility but 7 and 8 in the first and last digit, respectively. This way, we guarantee that the second, third, and fourth digits are the only digits have to worry about the number of possible PIN codes:

Let M be the number of possible codes that starts with 7 and ends with 8. Therefore:

M = 1 * 10 * 10 * 10 * 1

M = 1000 PIN numbers

Step 3: Finalize.

Then, we have to put the two numbers in rational form. Out of 90,000 PIN numbers that are available, there are only 1000 which starts at 7 and ends with 8. In numeric form, it looks like:

P = [tex]\frac{1000}{90000}[/tex]

Then, switch them to lowest terms:

P = [tex]\frac{1}{90}[/tex]

Therefore, there is a [tex]\frac{1}{90}[/tex] chance that the PIN code begins with a 7 and ends with an 8.

19. How many 4-letter codes can be formed with the letters M, N, O, P,Q, R, and S with and without repetition?


Answer:

840

Step-by-step explanation:

The problem itself is known as a permutation.

Permutation is a mathematical calculation of the number of ways a particular set can be arranged.

The formula for permutation is:

P(n,r)= n!/(n-r)!

There are 7 letters.

n=7

r=4

P( 7,4)=7!/(7-4)!

P(7,4)=7!/3!

P(7,4)=7×6×5×4×3!/3!

P(7,4)=7×6×5×4

P(7,4)=840

24. How many four-letter codes can be formed from the letters of the word MATHEMATICS if repetition of letters is not allowed?


Answer:

The Answer is 625

Step-by-step explanation:

For the first letter, there are 5 choices for letters, as it can be A, B, C, D, or E. Since repetition is allowed, there are also 5 choices for the second letter…

Every letter has five choices, so

5 ∗ 5∗ 5 ∗ 5= 54 = 625.

Why did we multiply? Let’s take an example where the question asks only about 2 letter words. When the 1st letter is A, we have 5 possibilities for the 2nd letter. When the 1st letter is B, we also have 5 for the 2nd… So in this case, it would be 5*5.

The same applies for 4 letter words, we just have more spaces. Again, 5 possibilities for each letter space means 5 ∗ 5 ∗ 5 ∗ 5= 54 = 625.
