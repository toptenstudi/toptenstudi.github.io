3. A multiple-choice quiz has 8 questions, with 4 possible answers for each question. How many possible ways are there to complete the test?​


There are 4 possible answers for each question, so for each question, there are 4 options to choose from.

For the first question, you can choose 1 of 4 options.
For the second question, you can choose 1 of 4 options.
And so on, for all 8 questions.

So, the number of ways to complete the test is:
4 * 4 * 4 * 4 * 4 * 4 * 4 * 4 = 4^8 = 65,536

Therefore, there are 65,536 possible ways to complete the test.

7. Topic: ProbabilityA teacher gave a quiz consisting of 5 item matching test and a 5 item multiple choice with 4 choices each. If a student answer each question in the quiz, in how many ways can you answer the question?Please pasagot po ng tama​


Answer:

122,880

Step-by-step explanation:

For the first part, there are 5 items to match and 5 choices. We can choose the first item in 5 ways, then the second item in 4 ways (since one choice has already been used), then the third item in 3 ways, then the fourth item in 2 ways, and the last item in 1 way. So the total number of ways to match the items is:

5 x 4 x 3 x 2 x 1 = 120

For the second part, there are 5 multiple choice questions with 4 choices each. For each question, we can choose one of the 4 choices, so the total number of ways to answer the multiple choice questions is:

4 x 4 x 4 x 4 x 4 = 4^5 = 1024

To find the total number of ways to answer the entire quiz, we multiply the number of ways to answer the matching questions by the number of ways to answer the multiple choice questions:

120 x 1024 = 122,880

Therefore, there are 122,880 ways to answer the quiz.

21. Topic: ProbabilityIf you are answering an 10 item multiple choice quiz with 4 choices per item and with only one correct answer, what is the probability that you will get?1. exactly 3 correct answer?2. fewer than 3 correct answer?3. atleast 1 correct answer?​


Answer:

To find the probability of getting exactly 3 correct answers, we need to use the binomial probability formula:

P(X = k) = C(n,k) * p^k * (1-p)^(n-k)

where

n = number of trials (number of questions in the quiz) = 10

k = number of successes (number of correct answers) = 3

p = probability of success (probability of getting one correct answer) = 1/4

Plugging in the values, we get:

P(X = 3) = C(10,3) * (1/4)^3 * (3/4)^7

P(X = 3) = 120 * 0.0039 * 0.1335

P(X = 3) = 0.6237%

Therefore, the probability of getting exactly 3 correct answers is 0.6237%.

To find the probability of getting fewer than 3 correct answers, we need to add up the probabilities of getting 0, 1, or 2 correct answers.

P(X < 3) = P(X = 0) + P(X = 1) + P(X = 2)

Using the same formula, we get:

P(X = 0) = C(10,0) * (1/4)^0 * (3/4)^10 = 0.0563%

P(X = 1) = C(10,1) * (1/4)^1 * (3/4)^9 = 0.5634%

P(X = 2) = C(10,2) * (1/4)^2 * (3/4)^8 = 2.5267%

Adding these probabilities up, we get:

P(X < 3) = 0.0563% + 0.5634% + 2.5267%

P(X < 3) = 3.1464%

Therefore, the probability of getting fewer than 3 correct answers is 3.1464%.

To find the probability of getting at least 1 correct answer, we need to find the probability of getting 0 correct answers and subtract it from 1.

P(X >= 1) = 1 - P(X = 0)

Using the binomial probability formula, we get:

P(X = 0) = C(10,0) * (1/4)^0 * (3/4)^10 = 0.0563%

Subtracting this from 1, we get:

P(X >= 1) = 1 - 0.0563%

P(X >= 1) = 99.9437%

Therefore, the probability of getting at least 1 correct answer is 99.9437%.

30. A student writes a five-question multiple-choice quiz. Each question has four possible responses. The student guesses at random for each question. Calculate the probability for each possible score on the test from 0 to 5.


Answer:

If the student's guesses are uniformly at random and the five guesses are independent of each other, then the probability would be:

(14)5=11024≈0.1%(14)5=11024≈0.1%

But the probability of those conditions being met is almost certainly even lower than that!

Any actual student would have at least some faint inkling that some choices were incorrect. Even if they were trying to be deliberately random, human beings are known to be rather poor random number generators, typically under-representing a guess like 2–2–2–2–2 and over-representing more "random looking" guesses like 4–1–3–2–4. Exam setters also do not allocate the correct answers uniformly at random.

Hence it is difficult to accurately assess a real-world probability, but I can safely say that in a student population of ten thousand just guessing answers, at least one student would get a perfect score
