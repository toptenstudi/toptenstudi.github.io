WHAT IS ASKED

This asks about the question or the problem in the situation. Usually, it asks of the number of something missing or asked in the word problem. It is written in a statement form like the example above.

WHAT ARE THE GIVEN FACTS

This tells the numbers given in the problem and what these numbers are all about.

WHAT OPERATION WILL YOU USE

This is about the operation used to solve the problem by using the numbers in the given facts. There are word problems that use more than one operation. Operations are addition, subtraction, multiplication and addition.

WHAT IS THE NUMBER SENTENCE

The number sentence is the formula or the statement using the numbers in the given facts including the operation used. Looking at the example above letter “N”is written after the equal sign. “N”represents number or the answer not yet revealed.

WHAT IS THE ANSWER

This part reveals the answer to the given problem following the steps shown in the number sentence.
