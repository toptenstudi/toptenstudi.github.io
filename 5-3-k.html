6. Find k. ( k + 3, 5 ) ( 1, k - 2 ) Slope = 4 Add solution pls!


Answer:

Sure!

Step-by-step explanation:

We can use the formula for the slope of a line passing through two points.

slope (y2-y1)/(x2-x1)

where (x1,y1) and (x2, y2) are the given points.

Plugging in the given values, we get:

4=(k-2-5)/(1-(k+3))

Simplifying the right-hand side, we get

4=(k-7)/(-k-2)

Cross-multiplying, we get

4k-8-k-7

Simplifying and solving for k, we get

5k=1

k=1/5

Therefore, the solution is k = 1/5.
