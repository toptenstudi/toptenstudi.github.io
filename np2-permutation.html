3. Q#14Find the value of n in the equation nP2 = 30 .​


[tex] \large\underline \bold{{SOLUTION}}[/tex]

[tex]\textsf{Using the permutation formula}[/tex]

[tex]\longmapsto\rm{nP2\implies P(n,r)= \frac{n!}{(n-r)!}}[/tex]

[tex]\longmapsto\rm{P(n,2)= \frac{n!}{(n-2)!}}[/tex]

[tex]\longmapsto\rm{30= \frac{n!}{(n-2)!}}[/tex]

[tex]\longmapsto\rm{30(n-2)!= n!}[/tex]

[tex]\text{Notice that n! , can be written as}[/tex][tex]\text{n(n-1)(n-2)!}[/tex]

[tex]\textsf{Now , We will solve for the equation:}[/tex]

[tex]\implies\sf{30(n-2)!= n!}[/tex]

[tex]\implies\sf{30(n-2)!=n(n-1)(n-2)!}[/tex]

[tex]\implies\sf{30\cancel{(n-2)!}=n(n-1)\cancel{(n-2)!}}[/tex]

[tex]\implies\sf{30=n(n-1)}[/tex]

[tex]\implies\sf{30=n²-n}[/tex]

[tex]\implies\sf{0=n²-n-30}[/tex]

[tex]\implies\sf{0=(n-6)(n+5)}[/tex]

[tex]\textsf{By zero-product property}[/tex]

n-6 = 0 , n = 6n+5 = 0 , n = -5

Ignoring the negative root , Hence the value of n is 6.

[tex] \large\underline{ \bold{ANSWER}}[/tex]

n = 6
