Formulate the mathematical equation.

(8.1+7.5+7.8+n)/4 = 7.7

Cross multiply

8.1 + 7.5 + 7.8 + n = 4(7.7)

23.4 + n = 30.8

n = 30.8 - 23.4

n = 7.4

a. Given: h = -16t² + 8t + 8

by factoring,

-16t² + 8t + 8 = 0

Factor the first term.

-16t² = (-4t)(4t)

Factor the last term.

8 = (4)(2)

Separate the factors and equate them to zero.

(-4t + 4)(4t + 2) = 0

-4t + 4 = 0 ; 4t + 2 = 0

Solve for t.

-4t + 4 = 0                          ;     4t + 2 = 0

-4t + 4 - 4 = 0 - 4               ;     4t + 2 - 2 = 0 - 2

-4t = -4                               ;     4t = -2

[tex]\frac{-4t}{-4}[/tex] = [tex]\frac{-4}{-4}[/tex]                              ;     [tex]\frac{4t}{4}[/tex] = [tex]\frac{-2}{4}[/tex]
