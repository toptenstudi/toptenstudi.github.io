5. Convert the following.1. 12 liters = _ mL2. 5 days = _ hours3. 15 feet = _ inches4. 3 meters = _ cm5. 6 hours = minutes​


CONVERSION

Convert the following.

12 liters = 12,000 milliliters5 days = 120 hours15 feet = 180 inches3 meters = 300 centimeters6 hours = 360 minutes

SOLUTION

1. 12 liters = ____ milliliters

There are 1,000 milliliters in 1 liter, so

[tex] \rm 12\:\cancel{ L } × \frac{ 1,000\:mL }{ 1\:\cancel{ L } } = \green { 12,000\:mL } [/tex]

2. 5 days = ____ hours

There are 24 hours in 1 day, so

[tex] \rm 5\:\cancel{ days } × \frac{ 24\:hours }{ 1\:\cancel{ day } } = \green { 120\:hours } [/tex]

3. 15 feet = ____ inches

There are 12 inches in 1 foot, so

[tex] \rm 15\:\cancel{ feet } × \frac{ 12\:inches }{ 1\:\cancel{ foot } } = \green { 180\:inches } [/tex]

4. 3 meters = ____ centimeters

There are 100 centimeters in 1 meter, so

[tex] \rm 3\:\cancel{ m } × \frac{ 100\:cm }{ 1\:\cancel{ m } } = \green { 300\:cm } [/tex]

4. 6 hours = ____ minutes

There are 60 minutes in 1 hour, so

[tex] \rm 6\:\cancel{ hours } × \frac{ 60\:minutes }{ 1\:\cancel{ hour } } = \green { 360\:minutes } [/tex]
