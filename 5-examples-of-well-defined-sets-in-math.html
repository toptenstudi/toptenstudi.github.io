2. Give 5 example of well defined sets math


Well-defined set is a set with an exact object or element.

A set is a well-defined collection of distinct objects. The objects of a set are called its elements. If a set has no elements, it is called the emptyset and is denoted by ∅. Note: ∅ is not the same as 0.

Example:

{a subject in Grade 7} - This is a well-defined set, because it is clear whether a subject is taught in Grade 7

1. {a character in the novel Noli Me Tangere}

2. {a national hero of the Philippines}

3. {a caffeinated drink}

4. {a country of Southeast Asia}

5. {a Greek mythology goddess}
