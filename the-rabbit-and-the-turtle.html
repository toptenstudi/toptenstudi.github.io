29. CHALLENGE PROBLEM * 1. A rabbit and a turtle are practicing for theirbig race. The rabbit covers a 30. m practice course in 5.0 seconds, the turtle covers the same distance in 120 seconds. If the actual raceis run on a 96 m course, by how many seconds will the rabbit beat the turtle?​


Explanation:

To solve this problem, we can first find the speed of the rabbit and the turtle:

Speed of rabbit = distance / time = 30 m / 5.0 s = 6.0 m/s

Speed of turtle = distance / time = 30 m / 120 s = 0.25 m/s

Next, we can use the speeds to find the time it takes for each animal to complete the actual race course:

Time for rabbit = distance / speed = 96 m / 6.0 m/s = 16 s

Time for turtle = distance / speed = 96 m / 0.25 m/s = 384 s

Finally, we can find the difference between the times to see how many seconds the rabbit will beat the turtle by:

Difference in time = time for turtle - time for rabbit = 384 s - 16 s = 368 s

Therefore, the rabbit will beat the turtle by 368 seconds, or about 6 minutes and 8 seconds
