12. Directions: Solve the following absolute value inequalities and choose the letter of the correct answer from the given choices.5. Solve the absolute value inequality: |1 – 2w| < 5 a. {c | 3 < c < –2} b. {c | –3 < c < 2} c. {c | 3 > c > –2} d. {c | –3 > c > 2}


Answer:

c. {c | 3 > c > –2}

Step-by-step explanation:

To solve for this problem, what we do first is to apply the Absolute rule that is:

[tex]\mathrm{If}\:|u|\:<\:a,\:a>0\:\mathrm{then}\:-a\:<\:u\:<\:a[/tex]

From here, we get: [tex]-5<1-2w<5[/tex] and then [tex]1-2w>-5\quad \mathrm{and}\quad \:1-2w<5[/tex]

Afterwards, we solve for the value of [tex]w[/tex] for each inequality. So:

[tex]1-2w>-5\\\mathrm{Subtract\:}1\mathrm{\:from\:both\:sides}\\\\[/tex]

[tex]1-2w-1>-5-1\\\mathrm{Simplify}\\-2w>-6\\\mathrm{Multiply\:both\:sides\:by\:-1\:\left(reverse\:the\:inequality\right)}\\\left(-2w\right)\left(-1\right)<\left(-6\right)\left(-1\right)\\\mathrm{Simplify}\\\\[/tex]

[tex]2w<6\\\mathrm{Divide\:both\:sides\:by\:}2\\\frac{2w}{2}<\frac{6}{2}\\\mathrm{Simplify}\\w<3[/tex]

Then, for the other we inequality, we do the same as the first inequality. So:

[tex]1-2w<5\\\mathrm{Subtract\:}1\mathrm{\:from\:both\:sides}\\1-2w-1<5-1\\\mathrm{Simplify}\\-2w<4\\Multiply\:both\:sides\:by\:-1\:\left(reverse\:the\:inequality\right)\\\left(-2w\right)\left(-1\right)>4\left(-1\right)\\\mathrm{Simplify}\\2w>-4\\\mathrm{Divide\:both\:sides\:by\:}2\\\frac{2w}{2}>\frac{-4}{2}\\\mathrm{Simplify}\\w>-2\\[/tex]

Finally, we can combine the intervals as:

[tex]w<3\quad \mathrm{and}\quad \:w>-2 \, : \, -2<w<3[/tex]

Rearranging the answer, we can say that the answer can be transformed as [tex]3 > w > -2[/tex] and finally putting it into the answer's format, we get:

c. {c | 3 > c > –2}
