2. The limits at infinity of polynomial functions of degree and leading coefficient . If n = ________


Answer:

If n is the degree of a polynomial function and a is the leading coefficient, then the limits at infinity of the polynomial function are determined by the sign of a and the parity of n.

If n is even, then the limits at infinity are the same as the limit as x approaches infinity of the term with the highest power. That is, if a > 0, the limit is infinity, and if a < 0, the limit is negative infinity.

If n is odd, then the limits at infinity are the same as the limit as x approaches infinity of the term with the highest power and the sign of a. That is, if a > 0, the limit is infinity for x going to infinity and negative infinity for x going to negative infinity, and if a < 0, the limit is negative infinity for x going to infinity and infinity for x going to negative infinity.

So, n is the degree of the polynomial function.

- If the leading coefficient is positive, then the limit as x approaches infinity is positive infinity.

- If the leading coefficient is negative, then the limit as x approaches infinity is negative infinity.
