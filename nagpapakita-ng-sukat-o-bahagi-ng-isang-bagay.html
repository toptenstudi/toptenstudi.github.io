Answer:

Ang graph ay paraan para madaling maunawaan at mabigyang kahulugan ang mga impormasyon. Ang graphay ginagamitan ng mga simbolo.

Mga Uri ng Graph

1. Line Graph (Talangguhit)

Ginagamit upang ipakita ang iba’t ibang impormasyon at datos sa pamamagitan ng paggamit ng mga linya. Ito ay ginagamit upang ipakita ang pagtaas at pagbaba ng datos samagkaibang panahon o pangyayari.

2. Bar Graph

Ginagamit sa paghahambing o pagpapakita ng kalakaran ng sukat. Nagpapakita ito ng kaugnayan ng mga ideya o paghahambing ng magkaugnay na mga ideya.

3.. Pie Graph

Ang paggamit ng pie graph ay ganap na popular. Tulad ng bilog, ito ay nagbibigay ng biswal na konsepto ng buo (100%).

4. Picture Graph o Pictograph

Ito ang graph na gumagamit ng mga larawan.
