4. a new building has 4455 bricks when finish .If the builder allows. 120 extra bricks for breakage ,how many bricks will he need to build 7 building ​


He will need 32,025 bricks to build 7 buildings.

Step-by-Step Explanation:

Word Problem

The problem is asking for the total number of bricks that the builder would need to build 7 buildings. There are two steps to find the answer to the problem.

Add the needed bricks to finish one building and the extra bricks for breakage.Multiply the total number of bricks needed in one building by 7, since the builder has to build 7 buildings.

Given:

bricks to finish one building - 4,455

extra bricks - 120

number of buildings - 7

Equation:

(4,455 + 120) × 7 = N

Solution:

Step 1: Add

4,455 + 120 = 4,575

Step 2: Multiply

4,575 × 7 = 32,025

Final Answer:

32,025 bricks

19. a new building has 4455 bricks when finished


Answer:

He will need 32,025 bricks to build 7 buildings.

Explanation:

Word Problem

The problem is asking for the total number of bricks that the builder would need to build 7 buildings. There are two steps to find the answer to the problem.

1. Add the needed bricks to finish one building and the extra bricks for breakage.

2. Multiply the total number of bricks needed in one building by 7, since the builder has to build 7 buildings.

Given:

bricks to finish one building - 4,455

extra bricks - 120

number of buildings - 7

Equation:

(4,455 + 120) × 7 = N

Solution:

Step 1: Add

4,455 + 120 = 4,575

Step 2: Multiply

4,575 × 7 = 32,025

Final Answer:

32,025 bricks
