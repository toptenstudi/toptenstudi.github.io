17. There are 20%, percent more goblins than wizards in magic club. There are 120 goblins in magic club. How many wizards are in the magic club?


Answer:

There are 100 wizards in the magic club.

Step-by-step explanation:

Let us first assign variables to the given and unknown:

Let x = number of goblins

Let y = number of wizards

We then write out an equation using these variables:

[tex]x = y + 0.20y[/tex] (Since the number of goblins is 20% or 0.20 more than the number of wizards)

[tex]x = 120[/tex] (as given, there are 120 goblins in magic club)

[tex]120 = y + 0.20y[/tex]

[tex]120 = 1.20y[/tex]

[tex]y = \frac{120}{1.20}[/tex]

[tex]y = 100[/tex]

Hence, there are 100 wizards in the magic club.
