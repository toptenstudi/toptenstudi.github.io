21. 1. if 15 × 5 = 114, which number base has been used? 2. solve the equation 120n=63{16}, where n and 16 bases?


Answer:

1. It is not possible for 15 x 5 to equal 114 in any number base, including non-standard bases. This is because in any base, the product of two single-digit numbers cannot exceed the largest single-digit number in that base multiplied by itself (for example, in base 10, the largest single-digit number is 9, so the product of two single-digit numbers cannot exceed 81).

2. To solve the equation 120n = 63{16} for n in base 16, we first need to convert 120 to base 16. To do this, we can repeatedly divide 120 by 16 and write the remainders in reverse order. We get:

120 ÷ 16 = 7 remainder 8

120 ÷ 16 = 7 remainder 8 7 ÷ 16 = 0 remainder 7

So in base 16, 120 = 78. Next, we need to convert 63{16} to base 10. We can do this by expanding the number in the usual way:

63{16} = 6 x 16^1 + 3 x 16^0 = 96 + 3 = 99

So the equation becomes:

78n = 99

Dividing both sides by 78, we get:

n = 1{12}

Therefore, the solution to the equation 120n = 63{16} in base 16 is n = 1{12}.
