14. add the following integers using algebra tiles counters


This problem can be represented with color counters/tiles. In this case, the red counters represent positive numbers and the black ones represent the negative numbers.

One positive counter and one negative counter/tile equals zero because 1+(−1)=0. Draw a line through the counters that equal zero.

The remaining counters represent the answer. Therefore, 4+(−2)=2. The answer is the difference between 4 and 2. The answer takes on the sign of the larger number. In this case, the four has a positive value and it is greater than 2.
