Each of these 20 different possible selections is called a permutation. In particular, they are called the permutations of five objects taken two at a time, and the number of such permutations possible is denoted by the symbol 5P2, read “5 permute 2.” In general, if there are n objects available from which to select, and permutations (P) are to be formed using k of the objects at a time, the number of different permutations possible is denoted by the symbol nPk. A formula for its evaluation is

nPk = n!/(n − k)!

The expression n!—read “n factorial”—indicates that all the consecutive positive integers from 1 up to and including n are to be multiplied together, and 0! is defined to equal 1. For example, using this formula, the number of permutations of five objects taken two at a time is

18. If an experiment consists of throwing a die and then drawing a letter at random from the English alphabet, how many points are in the sample space?


The number of points in the sample space is 156. To get the sample space, you must get the number of possibilities there is with the given situation. There are 6 sides in a dice and 26 letters in the alphabet. The short method is to multiply 6 and 26 to get sample space.

Set S = 6 × 26

Set S = 156

Background

A sample space is the set of the probability of the experiment. The problem asks for the number of possibilities considering the sides of the dice and the letters of the alphabet. Simply: “How many results are there for one side and one letter to come out together?”

Solution (Long Method) Identify the sample of the two different sets. Combine the two sets. Count the number of elements or members in the set.  Answer.  

The sample set of the dice is {1, 2, 3, 4, 5, 6}.  

The sample set of the alphabet is {A, B, C, …, X, Y, Z}

The sample set of the dice and the alphabet is {1A, 1B, 1C, …, 2A, 2B, 2C, …, 5A, 5B, 5C, …, 6A, 6B, 6C}.

The sample set has 156 elements. Therefore, the sample space has 156 points.
