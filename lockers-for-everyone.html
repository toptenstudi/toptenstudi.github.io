7. My math book is in my locker.All the digits in my locker are odd numbers and their sum is 11 there are 120 lockers. What is my locker number?


Since there are 120 lockers that would mean that the locker number has 3 digits (including 0 as the first digit). Since all the digits are odd numbers we let the first digit be 2a - 1, the second be 2b - 1 and the third be 2c - 1.

(2a - 1) + (2b - 1) + (2c - 1) = 11
2(a+b+c) - 3 = 11
2(a+b+c) = 14
a + b + c = 7

We need to take note that the first digit can only be 0 or 1. Since all digits are odd a is 1.

1 + b + c = 7
b + c = 6

The possible pairs are (1,5), (2,4), (3,3), (4,2), and  (5,1). Since all digits are odd we only take the pairs (1.5), (3,3) and (5,1). The only possible locker numbers are from 100 to 120 since our first digit is 1. So we can only take the pair (1,5).

Therefore the locker number is 115.

answer:

Using Polya's four steps in problem solving, we can solve this problem as follows:

Understand the problem: In this problem, there are 1,000 lockers in a high school and 1,000 students. Each student takes a turn, starting with the first student who opens all 1,000 lockers, and the second student closes every other locker, and so on. The question is asking how many lockers will be open at the end.

Devise a plan: We can solve this problem by determining the factors of each locker number and seeing if the number of factors is even or odd. If the number of factors is even, the locker will be closed at the end. If the number of factors is odd, the locker will be open at the end.

Carry out the plan: After determining the factors of each locker number, we can see which lockers have an odd number of factors and which have an even number of factors. Only locker numbers that are perfect squares will have an odd number of factors. So, we can find out how many perfect squares are there between 1 and 1000.

Look back: Since the lockers are numbered from 1 to 1000, there are 31 perfect squares between 1 and 1000. Therefore, 31 lockers will be open at the end.
