2. why do switches flood frames addressed to unknown destinations?​


Answer:

Since both the multicast and broadcast addresses are never used in the source address field of a frame, a switch never learns these addresses. And since a switch never learns these addresses, it always floods the frames which have these addresses as the destination address.

3. What is Address Resolution Protocol(ARP)?


Answer:

The address resolution protocol (arp) is a protocol used by the Internet Protocol (IP) [ RFC826] , specifically IPv4, to map IP network addresses to the hardware addresses used by a data link protocol.
