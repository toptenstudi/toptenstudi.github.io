1. How many possible sample can be drawn from the population​


Step-by-step explanation:

The number of possible samples that can be drawn from a population depends on the size of the population and the size of the sample. The formula for calculating the number of possible samples is:

nCr = n! / r! * (n - r)!

Where n is the size of the population and r is the size of the sample.

For example, if we have a population of 10 and we want to draw a sample of 3, the number of possible samples would be:

10C3 = 10! / 3! * (10-3)! = 120

Therefore, there are 120 possible samples of size 3 that can be drawn from a population of 10.

29. There are nine identified points on a plane. How many possible triangle can be drawn?​


Answer:

In general the number of triangles which can be drawn from N points is:

(N)(N-1)(N-2)/6 where N>2 and the points are the vertices of the triangles.

Thus when N = 9, the number of such triangles a simpler question:

How many triangles can be created by lines drawn between 6 non-collinear points?

If the points are disposed in an array corresponding to the vertices of a regular hexagon, then 20 triangles having those points as triangle vertices are formed. However, the actual number of triangles formed is 110 (maybe more if I’ve miscounted).

It follows that the number of triangles formed from 9 points will be far greater than 84.
