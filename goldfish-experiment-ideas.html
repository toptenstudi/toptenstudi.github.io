Answer:

Let's call the number of goldfish in the pond "x". So, the ratio of goldfish to koi is 5.2, meaning there are 5.2 times as many goldfish as koi. We know there are 8 koi, so:

x/8 = 5.2

So, we can solve for x:

x = 5.2 * 8 = 41.6

So, there are 41.6 goldfish in the pond.

4. how to breed goldfish​


To breed goldfish, start by placing a male and female goldfish in the same tank.

Then, wait for the female to drop her eggs and the male to fertilize them.

Once you notice the fertilized eggs in the tank, transfer both goldfish to a different tank so they don't eat the eggs. Then, wait 4-7 days for the eggs to hatch.

7. There are 725 carps and some goldfish in an aquarium. There are five times as many carps as goldfish. How many fishes are there in the aquarium?


Answer:

Let's call the number of goldfish "x."

According to the problem, there are five times as many carps as goldfish. So the number of carps is 5x.

We also know that the total number of fish is 725.

Putting this all together, we can write the equation:

x + 5x = 725

Simplifying, we get:

6x = 725

Dividing both sides by 6, we get:

x = 120.83

Since we can't have a fraction of a fish, we need to round this down to the nearest whole number. So there are 120 goldfish.

To find the number of carps, we can use the equation we came up with earlier:

carps = 5x

carps = 5(120)

carps = 600

So there are a total of 120 + 600 = 720 fish in the aquarium.

Answer:

Step-by-step explanation:

Celia bought a bag of 12 goldfish for $3. What is the cost of 1 goldfish?

Let x be the cost of each goldfish:

[tex]12:3=1:x \\ \\ \frac{12}{3} = \frac{1}{x} \\ \\ (12)(x) = (1)(3) \\ \\ 12x = 3 \\ \\ \frac{12x}{12} = \frac{3}{12} \\ \\ x = 0.25[/tex]
